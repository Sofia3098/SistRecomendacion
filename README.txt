Sistema de Recomendación

Presentado por: Peñares Pillaca, Sofia Fernanda

Los códigos en esta presentación son los siguientes:
- LimpDisp.py
    Código donde se muestra la dispersión de la data sin limpiar y su limpieza propia, para después poder mostrar
    dispersión con el dataset limpio
- SistRemcom.py
    Código con el los siguientes sistemas de recomendación:
    * Distancia de Manhattan
    * Distancia Euclidiana
    * Correlación de Pearson
    * Similitud del coseno
    implementando un KNN para poder obtener los vecinos más cercanos de recomendación
- SistRecomMatDisp.py
    Código con los sistemas de recomendación antes mencionados y con el sistema KNN, pero agregando la esctructura
    de datos de 'Matrices de Dispersión' para la optimización y escalabilidad del código.
